ASCII_ART_GRAYSCALE
Grayscale Image From Printable Characters

ASCII_ART_GRAYSCALE is a data directory which contains examples of "ASCII Art" which uses printable characters to form a grayscale image.

Related Data and Programs:

MANDELBROT_ASCII, a C program which generates an ASCII image of the Mandelbrot fractal set using just two lines of C.

Sample Files:

  • grayscale_10_levels.txt, a sequence of 10 characters that can be used for a grayscale ASCII art image.
  • grayscale_70_levels.txt, a sequence of 70 characters that can be used for a grayscale ASCII art image.
  • mandelbrot_ascii.txt, an image of the Mandelbrot set, created by the program MANDELBROT_ASCII.
  • marilyn_monroe.txt.
  • mona_lisa.txt.
