MET Files
a Metafile Format for the DISLIN Graphics Package

MET is a data directory which contains examples of files in the MET format, a metafile format for the DISLIN graphics package.

Graphics images created by DISLIN can be stored symbolically in a MET file. DISLIN can retrieve this data, and display it to the screen, or convert it to any of a number of standard graphics file formats.

Users desiring a specific graphics file format from DISLIN call the function metafl(). One of the input options to this function is "GKSL", which creates a GKSLIN metafile. If metafl() is not called, the default format is apparently GKSLIN, so I am presuming that this format is what is meant when DISLIN refers to its metafile format, although DISLIN also makes available a DISLIN "IMAG" format as well.
