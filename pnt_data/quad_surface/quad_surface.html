QUAD_SURFACE
Surfaces Defined by Quadrilateral Mesh Data

QUAD_SURFACE is a data directory which contains examples of QUAD_SURFACE data, defining a surface in 3D, using a mesh of quadrilaterals. The format requires three files:

  • the node file lists the 2D coordinates of a set of points.
  • the quadrilateral file lists sets of 4 node indices, forming the quadrilaterals.
  • the value file lists the Z value, or height, or some other quantity, associated with each node.

Related Data and Programs:

POLYGONAL_SURFACE, a data directory which contains examples of polygonal surface files.

QUAD_SURFACE_DISPLAY, a MATLAB program which displays an image of the surface defined by a quadrilateral mesh.

TABLE, a format which is used for the files.

TRI_MESH_DISPLAY, a MATLAB program which reads files defining a triangulated surface and displays an image.

Sample Files:

POOL is depth data for a swimming pool.

  • pool_node_xy.txt, the coordinates of the 28 nodes;
  • pool_quad_node.txt, the 18 quadrilaterals formed from the nodes;
  • pool_value_xy.txt, the pool depth measured at each node;
  • pool.png, a PNG image of the surface created by QUAD_MESH_DISPLAY.
