VTS Files

VTS is a data directory which contains examples of VTS files, a file format used by the Visualization Toolkit, which contains information associated with a curvilinear or structured grid, which can be displayed by the paraview program;

PARAVIEW is a 3D graphics program which can read VTS files and display the data. The home page for PARAVIEW is http://www.paraview.org/ .

PARAVIEW, examples which illustrate the use of the paraview graphics program.

VTK, a data directory which contains examples of legacy VTK files, a file format used by the Visualization Toolkit, and which can be displayed by the paraview program;
