POLY_3D Files

POLY_3D is a data directory which contains examples of 3D POLY files, a format used by tetgen and tetview to define a Piecewise Linear Complex (PLC).

  • pmdc.node, the level 0 node file;
  • pmdc.poly, the level 0 poly file;
